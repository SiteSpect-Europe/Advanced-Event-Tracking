SPA Adapter

No test written yet. This module patches History's 'pushState', 'replaceState', 'back' and 'go' method. Should work for modern SPA's.

When SPA adapter is included, non-instant events with match / filter will be evaluated every page change. 
		_stsp.push({
			'event': 'ReachedContactPage',
			'match': {
				'pathName': 'contact'
			}
		})

		_stsp.push({
			'event': 'ReachedContactPage',
			'filter': function(){
				return somecondition === true
			}
		})
